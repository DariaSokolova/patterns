Interpreter

Description

Given a language, defines a represention for its grammar along with an interpreter that uses the representation to interpret sentences in the language.
Should be used when the grammar is simple; efficiency is not a critical concern.

Using in java API:

java.util.Pattern
java.text.DateFormat

Pros:

  • Easy to change and extend the grammar.
  • Implementing the grammar is easy.
  • Makes it easier to evaluate an expression in anew way.

Cons:

  • Complex grammars are hard to maintain.
